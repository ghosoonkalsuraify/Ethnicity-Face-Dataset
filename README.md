# Ethnicity Classification Dataset (CNN + CapsNet)

## 📄 Paper

Race Classification by Facial Features Using CNN and Capsule Networks
https://doi.org/10.18280/isi.301209

## 📊 Dataset

This dataset contains 3 ethnic groups:

* African (1130 images)
* Asian (1140 images)
* Latino (1050 images)

All images were:

* Cropped using CNN-based face detection
* Resized to 128×128
* Augmented (rotation, scaling, flipping, brightness, contrast)

## 📥 Download Dataset

Available on Kaggle:

https://www.kaggle.com/datasets/alghosoonkar/ethnicity-face-dataset

## 🧠 Method

* CNN for face detection
* Capsule Network for classification

## 📈 Results

* African: 98%
* Asian: 97%
* Latino: 97%
